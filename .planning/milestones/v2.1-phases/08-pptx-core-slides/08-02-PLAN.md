**Exported pure data-mapping helpers** (testable without pptxgenjs):

1. `buildTitleSlideData(store: ReturnType<typeof useInputStore>)`:
   Returns `{ deploymentMode: string, date: string }` where date is `new Date().toISOString().split('T')[0]`.

2. `buildConfigSummaryData(store: ReturnType<typeof useInputStore>)`:
   Returns `Array<{ label: string, value: string }>` with exactly these 8 rows:
   - Hosts: `String(store.hostCount)`
   - Cores per socket: `String(store.coresPerSocket)`
   - Sockets per host: `String(store.socketsPerHost)`
   - RAM per host: `${store.hostRamGB} GB`
   - Storage per host: `${store.hostStorageTB} TB`
   - Storage type: `store.storageType`
   - Network speed: `${store.networkSpeedGbE} GbE`
   - Management architecture: `store.managementArchitecture`

3. `buildWorkloadSlideData(store: ReturnType<typeof useInputStore>)`:
   Returns `Array<{ label: string, value: string }>` with exactly these 6 rows:
   - VM count: `String(store.vmCount)`
   - vCPU per VM: `String(store.avgVcpuPerVm)`
   - vRAM per VM: `${store.avgVramGbPerVm} GB`
   - Storage per VM: `${store.avgStorageGbPerVm} GB`
   - CPU overcommit ratio: `${store.cpuOvercommitRatio}:1`
   - RAM overcommit ratio: `${store.ramOvercommitRatio}:1`

4. `buildMgmtOverheadData(mgmt: MgmtDomainResult)`:
   Import `MgmtDomainResult` from `@/engine/types` (type-only import is fine).
   Returns `Array<{ label: string, cores: number, ramGB: number }>` with exactly 6 rows:
   - `{ label: 'vCenter', cores: mgmt.vcenterCores, ramGB: mgmt.vcenterRamGB }`
   - `{ label: 'SDDC Manager', cores: mgmt.sddcCores, ramGB: mgmt.sddcRamGB }`
   - `{ label: 'NSX', cores: mgmt.nsxCores, ramGB: mgmt.nsxRamGB }`
   - `{ label: 'Aria Ops / Fleet Mgr', cores: mgmt.opsCores, ramGB: mgmt.opsRamGB }`
   - `{ label: 'Automation', cores: mgmt.automationCores, ramGB: mgmt.automationRamGB }`
   - `{ label: 'Total', cores: mgmt.totalCores, ramGB: mgmt.totalRamGB }`

5. `buildComputeResultsData(compute: ComputeResult)`:
   Import `ComputeResult` from `@/engine/types`.
   Returns `{ recommendedHostCount: number, coreUtilizationPct: number, ramUtilizationPct: number, availableCores: number, availableRamGB: number, minHostsForCpu: number, minHostsForRam: number }` — direct passthrough of the relevant fields.

6. `buildStorageResultsData(storage: StorageResult)`:
   Import `StorageResult` from `@/engine/types`.
   Returns `{ rawCapacityTB: number, usableAfterRaidTB: number, lfsOverheadTB: number, metadataOverheadTB: number, safeUsableCapacityTB: number, raidScheme: string }` — direct passthrough.

7. `buildRecommendationsData(store: ReturnType<typeof useInputStore>, calc: ReturnType<typeof useCalculationStore>)`:
   Returns `string[]` with these recommendation lines:
   - `Recommended host count: ${calc.compute.recommendedHostCount}`
   - `Safe usable storage: ${calc.storage.safeUsableCapacityTB.toFixed(2)} TB`
   - `CPU utilization: ${calc.compute.coreUtilizationPct.toFixed(1)}%`
   - `RAM utilization: ${calc.compute.ramUtilizationPct.toFixed(1)}%`
   - If `calc.validationErrors.length > 0`: `Active warnings: ${calc.validationErrors.length}`

```typescript
export async function generatePptxReport(): Promise<void> {
  const store = useInputStore()
  const calc = useCalculationStore()

  // Dynamic import — PPTX-15: Vite code-splits automatically
  const PptxGenJS = (await import('pptxgenjs')).default
  // Fresh instance per export — Pitfall 6
  const pres = new PptxGenJS()
  pres.layout = 'LAYOUT_WIDE'  // 13.33 x 7.5 inches

  // MUST call before any addSlide() — Pitfall 2
  pres.defineSlideMaster({
    title: MASTER_NAME,
    background: { color: PPTX_MASTER_COLOR },
    objects: [
      { rect: { x: 0, y: 6.8, w: '100%', h: 0.7, fill: { color: PPTX_FOOTER_COLOR } } },
      { text: { text: 'VMware by Broadcom  |  VCF Sizer', options: {
        x: 0.3, y: 6.9, w: 9, h: 0.4, color: PPTX_LIGHT_TEXT, fontSize: 9,
      } } },
    ],
    slideNumber: { x: 12.5, y: 6.9, color: PPTX_LIGHT_TEXT, fontSize: 9 },
  })

  const titleData = buildTitleSlideData(store)
  const configData = buildConfigSummaryData(store)
  const workloadData = buildWorkloadSlideData(store)
  const mgmtData = buildMgmtOverheadData(calc.management)
  const computeData = buildComputeResultsData(calc.compute)
  const storageData = buildStorageResultsData(calc.storage)
  const recsData = buildRecommendationsData(store, calc)

  // Slide 1: Title (PPTX-03)
  const s1 = pres.addSlide({ masterName: MASTER_NAME })
  s1.addText('VCF 9.x Sizing Report', { x: 0.5, y: 1.5, w: 12, h: 1.5, fontSize: 36, bold: true, color: PPTX_WHITE })
  s1.addText(`Deployment: ${titleData.deploymentMode}`, { x: 0.5, y: 3.2, w: 12, h: 0.8, fontSize: 20, color: PPTX_LIGHT_TEXT })
  s1.addText(`Generated: ${titleData.date}`, { x: 0.5, y: 4.0, w: 12, h: 0.6, fontSize: 14, color: PPTX_LIGHT_TEXT })

  // Slide 2: Configuration Summary (PPTX-04)
  const s2 = pres.addSlide({ masterName: MASTER_NAME })
  s2.addText('Configuration Summary', { x: 0.5, y: 0.3, w: 12, h: 0.8, fontSize: 24, bold: true, color: PPTX_WHITE })
  const configRows = [
    [{ text: 'Parameter', options: { bold: true, fill: PPTX_HEADER_BG, color: '000000' } },
     { text: 'Value', options: { bold: true, fill: PPTX_HEADER_BG, color: '000000' } }],
    ...configData.map(r => [r.label, r.value]),
  ]
  s2.addTable(configRows, { x: 0.5, y: 1.3, w: 12, colW: [6, 6], rowH: 0.45, fontSize: 13, color: 'F0F0F0', border: { type: 'solid', pt: 1, color: '444444' } })

  // Slide 3: Workload Profile (PPTX-05)
  const s3 = pres.addSlide({ masterName: MASTER_NAME })
  s3.addText('Workload Profile', { x: 0.5, y: 0.3, w: 12, h: 0.8, fontSize: 24, bold: true, color: PPTX_WHITE })
  const wkldRows = [
    [{ text: 'Parameter', options: { bold: true, fill: PPTX_HEADER_BG, color: '000000' } },
     { text: 'Value', options: { bold: true, fill: PPTX_HEADER_BG, color: '000000' } }],
    ...workloadData.map(r => [r.label, r.value]),
  ]
  s3.addTable(wkldRows, { x: 0.5, y: 1.3, w: 12, colW: [6, 6], rowH: 0.45, fontSize: 13, color: 'F0F0F0', border: { type: 'solid', pt: 1, color: '444444' } })

  // Slide 4: Management Domain Overhead (PPTX-06)
  const s4 = pres.addSlide({ masterName: MASTER_NAME })
  s4.addText('Management Domain Overhead', { x: 0.5, y: 0.3, w: 12, h: 0.8, fontSize: 24, bold: true, color: PPTX_WHITE })
  const mgmtRows = [
    [{ text: 'Component', options: { bold: true, fill: PPTX_HEADER_BG, color: '000000' } },
     { text: 'vCPU', options: { bold: true, fill: PPTX_HEADER_BG, color: '000000' } },
     { text: 'RAM (GB)', options: { bold: true, fill: PPTX_HEADER_BG, color: '000000' } }],
    ...mgmtData.map(r => {
      const isTotal = r.label === 'Total'
      const opts = isTotal ? { bold: true } : {}
      return [{ text: r.label, options: opts }, String(r.cores), String(r.ramGB)]
    }),
  ]
  s4.addTable(mgmtRows, { x: 0.5, y: 1.3, w: 12, colW: [5, 3.5, 3.5], rowH: 0.45, fontSize: 13, color: 'F0F0F0', border: { type: 'solid', pt: 1, color: '444444' } })

  // Slide 5: Compute Results (PPTX-07)
  const s5 = pres.addSlide({ masterName: MASTER_NAME })
  s5.addText('Compute Results', { x: 0.5, y: 0.3, w: 12, h: 0.8, fontSize: 24, bold: true, color: PPTX_WHITE })
  const compRows = [
    [{ text: 'Metric', options: { bold: true, fill: PPTX_HEADER_BG, color: '000000' } },
     { text: 'Value', options: { bold: true, fill: PPTX_HEADER_BG, color: '000000' } }],
    ['Recommended Host Count', { text: String(computeData.recommendedHostCount), options: { bold: true } }],
    ['Min Hosts for CPU', String(computeData.minHostsForCpu)],
    ['Min Hosts for RAM', String(computeData.minHostsForRam)],
    ['Available vCPU', String(computeData.availableCores)],
    ['Available RAM (GB)', String(Math.round(computeData.availableRamGB))],
    ['CPU Utilization', `${computeData.coreUtilizationPct.toFixed(1)}%`],
    ['RAM Utilization', `${computeData.ramUtilizationPct.toFixed(1)}%`],
  ]
  s5.addTable(compRows, { x: 0.5, y: 1.3, w: 12, colW: [6, 6], rowH: 0.45, fontSize: 13, color: 'F0F0F0', border: { type: 'solid', pt: 1, color: '444444' } })

  // Slide 6: Storage Results (PPTX-08)
  const s6 = pres.addSlide({ masterName: MASTER_NAME })
  s6.addText('Storage Results', { x: 0.5, y: 0.3, w: 12, h: 0.8, fontSize: 24, bold: true, color: PPTX_WHITE })
  const storRows = [
    [{ text: 'Metric', options: { bold: true, fill: PPTX_HEADER_BG, color: '000000' } },
     { text: 'Value', options: { bold: true, fill: PPTX_HEADER_BG, color: '000000' } }],
    ['RAID Scheme', storageData.raidScheme],
    ['Raw Capacity', `${storageData.rawCapacityTB.toFixed(2)} TB`],
    ['Usable After RAID', `${storageData.usableAfterRaidTB.toFixed(2)} TB`],
    ['LFS Overhead', `${storageData.lfsOverheadTB.toFixed(2)} TB`],
    ['Metadata Overhead', `${storageData.metadataOverheadTB.toFixed(2)} TB`],
    [{ text: 'Safe Usable Capacity', options: { bold: true } }, { text: `${storageData.safeUsableCapacityTB.toFixed(2)} TB`, options: { bold: true } }],
  ]
  s6.addTable(storRows, { x: 0.5, y: 1.3, w: 12, colW: [6, 6], rowH: 0.45, fontSize: 13, color: 'F0F0F0', border: { type: 'solid', pt: 1, color: '444444' } })

  // Slide 7: Recommendations (PPTX-09)
  const s7 = pres.addSlide({ masterName: MASTER_NAME })
  s7.addText('Recommendations', { x: 0.5, y: 0.3, w: 12, h: 0.8, fontSize: 24, bold: true, color: PPTX_WHITE })
  const recText = recsData.map(r => `  \u2022  ${r}`).join('\n')
  s7.addText(recText, { x: 0.5, y: 1.5, w: 12, h: 4.5, fontSize: 16, color: PPTX_WHITE, valign: 'top', lineSpacingMultiple: 1.5 })

  // Trigger browser download — MUST await (Pitfall 5)
  await pres.writeFile({ fileName: 'vcf-sizing-report.pptx' })
}
```
