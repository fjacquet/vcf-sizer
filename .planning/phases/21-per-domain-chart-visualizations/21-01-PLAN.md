## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| store -> component | Reactive store data passed as props to chart components |
| Chart.js canvas -> uiStore | PNG data URLs captured from canvas and stored in memory |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-21-01 | Information Disclosure | chart PNG capture | accept | PNGs contain only sizing calculation data (no PII, no secrets); stored in ephemeral memory only, never persisted to disk or URL state |
| T-21-02 | Denial of Service | multiple Chart.js instances | accept | Bounded by number of workload domains (max ~10 in practice); Chart.js handles multiple instances natively |
| T-21-03 | Tampering | domain.id in canvas ID | accept | domain.id is a UUID generated by the app itself (not user input); used only as DOM element ID attribute |
</threat_model>
