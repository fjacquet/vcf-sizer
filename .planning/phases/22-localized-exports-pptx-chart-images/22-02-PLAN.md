```typescript
// Domain: Charts slide (EXPORT-01) -- only if chart images are available
const domainCharts = uiStore.chartImages[domain.id]
const chartTypes: Array<{ key: 'cores' | 'ram' | 'storage'; labelKey: string }> = [
  { key: 'cores', labelKey: 'export.cpuUtilization' },
  { key: 'ram', labelKey: 'export.ramUtilization' },
  { key: 'storage', labelKey: 'export.storageSizing' },
]
const availableCharts = chartTypes.filter(ct => domainCharts?.[ct.key])

if (availableCharts.length > 0) {
  const sCharts = pres.addSlide({ masterName: MASTER_NAME })
  sCharts.addText(`${domain.name} — ${t('export.charts')}`, {
    x: 0.5, y: 0.3, w: 12, h: 0.8,
    fontSize: 24, bold: true, color: PPTX_WHITE,
  })

  // Layout: up to 3 charts side by side on LAYOUT_WIDE (13.33 x 7.5)
  const positions = [
    { x: 0.3, y: 1.5, w: 4.0, h: 3.0 },
    { x: 4.5, y: 1.5, w: 4.0, h: 3.0 },
    { x: 8.7, y: 1.5, w: 4.0, h: 3.0 },
  ]

  availableCharts.forEach((ct, idx) => {
    const pos = positions[idx]
    sCharts.addImage({
      data: domainCharts![ct.key],  // full data:image/png;base64,... (PITFALL-7)
      x: pos.x, y: pos.y, w: pos.w, h: pos.h,
      altText: `${domain.name} ${ct.key} chart`,
    })
  })
}
```

Place `const uiStore = useUiStore()` at the top of `generatePptxReport()` alongside `store` and `calc` (not inside the loop).

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Chart data URLs | Base64-encoded PNG strings from Chart.js canvas -- generated client-side, no external input |
| pptxgenjs addImage | Accepts data URL and embeds in PPTX binary -- no network call |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-22-04 | Tampering | chartImages data URLs | accept | Data URLs are generated from Chart.js canvas.toDataURL() -- no user input vector. The canvas renders computed sizing data only. |
| T-22-05 | Information Disclosure | PPTX chart images | accept | Charts contain the same data visible in the UI -- no additional disclosure from embedding PNGs. |
| T-22-06 | Denial of Service | Large base64 images | accept | Each Chart.js canvas PNG is typically 20-50 KB base64. With 3 charts x N domains, PPTX size grows linearly but remains manageable (<5 MB for 10 domains). |
</threat_model>
